Introduction to C++

C++ is a powerful, high-performance programming language created by Bjarne Stroustrup as an extension of C. It combines the features of both high-level and low-level languages, making it versatile for system programming, game development, real-time simulations, and more. To start with C++, you need a C++ compiler, such as GCC or MSVC, and a text editor or an Integrated Development Environment (IDE) like Visual Studio, Code::Blocks, or CLion.

First C++ Program

Here's a basic "Hello, World!" program in C++:

                    
#include 

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
    Explanation:
  • #include <'iostream'>: This preprocessor directive includes the standard input-output stream library.
  • int main(): The main function where program execution begins.
  • std::cout: This is used to output text to the console.
  • std::endl: Ends the current line and flushes the output buffer.
C++ Comments

Comments are used to explain code and are ignored by the compiler. There are two types:

  • Single-line comments: Start with //.
  • Multi-line comments: Enclosed between /* and */.

Example: 

                    
// This is a single-line comment

/*
    This is a
    multi-line comment
*/
Keywords and Identifiers
  • Keywords: Reserved words with special meaning in C++, such as int, return, if, else. They cannot be used as names for variables or functions.
  • Identifiers: Names given to variables, functions, classes, etc. They must begin with a letter (uppercase or lowercase) or an underscore (_), followed by letters, digits, or underscores.
Variables, Literals and Constants
  • Variables: Named storage that can hold a value. For example, int age = 25; declares an integer variable named age with an initial value of 25.
  • Literals: Fixed values directly written in the code, like 42, 3.14, or 'A'.
  • Constants: Values that cannot be changed once assigned. Declared with the const keyword. Example: const int MAX = 100;.
Data Types

C++ has several basic data types:

  • 'int': Integer numbers.
  • 'float': Single-precision floating point numbers.
  • 'double': Double-precision floating point numbers.
  • 'char': Single character.
  • 'bool': Boolean value (true or false).
Type Modifiers

Type modifiers change the meaning of the base data types:

  • 'signed': Can hold positive and negative values.
  • 'unsigned': Can hold only non-negative values.
  • 'short': Shortened range of values.
  • 'long': Extended range of values.
  • 'bool': Boolean value (true or false).

Example: 

 
                    
unsigned int positiveNumber = 10;
long long int largeNumber = 123456789;
Constants

Constants are immutable values that are declared using 'const' or '#define' preprocessor directive.

Example: 

                    
const float PI = 3.14;
#define MAX_BUFFER_SIZE 1024
Basic Input/Output
  • Input: Use 'std::cin' to read input from the standard input (usually the keyboard).
    •                    
int age;
std::cin >> age;
  • Output: Use 'std::cout' to write output to the standard output (usually the console).
    •                 
std::cout << "Your age is " << age << std::endl;
Operators

Operators perform operations on variables and values:

  • Arithmetic Operators: '+' (addition), '-' (subtraction), '*' (multiplication), '/' (division), '%' (modulus).
  • Comparison Operators: '==' (equal), '!=' (not equal), '<' (less than), ''>' (greater than), '<=' (less than or equal to), '>=' (greater than or equal to).
  • Logical Operators: '&&' (logical and), '||' (logical or), '!' (logical not)
Conditionals

Conditionals control the flow of the program based on conditions. The main conditional statements are:

  • if: Executes a block of code if a specified condition is true.
  • else: Executes a block of code if the condition in the 'if' statement is false.
  • else if: Specifies a new condition if the previous one is false.
  • switch: Selects one of many blocks of code to be executed based on the value of a variable.

Example: 

                    
int number = 10;

if (number > 0) {
   std::cout << "Positive number" << std::endl;
} 
else if (number < 0) {
   std::cout << "Negative number" << std::endl;
} 
else {
   std::cout << "Zero" << std::endl;
}
Loops

Loops are used to repeat a block of code multiple times. The main types of loops are:

  • for: Repeats a block of code a specified number of times.
  • while: Repeats a block of code as long as a specified condition is true.
  • do while: Executes a block of code once, and then repeats as long as a specified condition is true.

Example: 

                    
// For loop
for (int i = 0; i < 5; i++) {
    std::cout << i << std::endl;
}
                        
// While loop
int j = 0;
while (j < 5) {
    std::cout << j << std::endl;
    j++;
}
                        
// Do-while loop
int k = 0;
do {
  std::cout << k << std::endl;
  k++;
} while (k < 5);
Functions

Functions are reusable blocks of code that perform a specific task. They can accept parameters and return a value.

  • Declaration: Specifies the function's return type, name, and parameters.
  • Definition: Contains the actual body of the function.
  • Call: Executes the function.

Example: 

                    
// Function declaration
int add(int a, int b);

// Function definition
int add(int a, int b) {
    return a + b;
}

// Function call
int sum = add(5, 3);
std::cout << "Sum: " << sum << std::endl;
Arrays in C++

Arrays in C++ are collections of elements of the same type stored in contiguous memory locations. They provide a way to store multiple values under a single name and access them using indices.

Declaration and Initialization

To declare an array, specify the type of its elements, the array name, and the size of the array in square brackets. You can also initialize the array at the time of declaration.

Example: 

                
int numbers[5] = {1, 2, 3, 4, 5};

Accessing Array Elements

Array elements are accessed using their indices, with the first element at index 0.

Example: 

                
#include 

int main() {
  int numbers[5] = {1, 2, 3, 4, 5};
                        
   // Access and print the first element
   std::cout << "First element: " << numbers[0] << std::endl;
                        
   // Modify the third element
   numbers[2] = 10;
                        
   // Access and print the modified third element
   std::cout << "Modified third element: " << numbers[2] << std::endl;
                        
  return 0;
}

Output:

                
First element: 1
Modified third element: 10

In this example:

  • 'numbers[0]' accesses the first element.
  • 'numbers[2]' accesses the third element and modifies its value to 10.
Pointers in C++

Pointers in C++ are variables that store the memory address of another variable. They provide a way to manipulate data in memory directly and are essential for dynamic memory allocation, passing arguments by reference, and working with arrays and data structures efficiently.

Declaration and Initialization

To declare a pointer, specify the type of the variable it will point to, followed by an asterisk ('*'), and then the pointer's name. Initialize the pointer using the address-of operator ('&').

Example: 

                
int num = 10;
int* ptr = #

This declares a pointer 'ptr' that stores the address of the integer variable 'num'.

Accessing and Modifying Values

Pointers can be dereferenced using the asterisk ('*') operator to access or modify the value at the memory address they store.

Example: 

                
#include 

int main() {
 int num = 10;
 int* ptr = #
                        
    // Access the value using the pointer
    std::cout << "Value of num: " << *ptr << std::endl;
                        
    // Modify the value using the pointer
    *ptr = 20;
                        
    // Access the modified value
    std::cout << "Modified value of num: " << num << std::endl;
                        
 return 0;
}

Output:

                
Value of num: 10
Modified value of num: 20

In this example:

  • '*ptr' is used to access the value stored at the memory address contained in 'ptr'
  • Modifying '*ptr' changes the value of 'num' because 'ptr' points to 'num'.

Null Pointers

A pointer can be initialized to 'nullptr' to indicate that it is not pointing to any valid memory location.

Example: 

                
int* ptr = nullptr;

Pointers and Arrays

When working with arrays, pointers can be used to traverse and manipulate the array elements.

Example: 

                
#include 

int main() {
 int numbers[5] = {1, 2, 3, 4, 5};
 int* ptr = numbers; // Points to the first element of the array

 for (int i = 0; i < 5; i++) {
   std::cout << *(ptr + i) << " "; // Access each element using the pointer
  }
                        
  return 0;
}

Output:

                
1 2 3 4 5

In this example:

  • 'ptr' points to the first element of the array 'numbers'.
  • '*(ptr + i)' accesses each array element by offsetting the pointer.

Pointers are powerful tools in C++ that provide flexibility and control over memory management and data manipulation.
